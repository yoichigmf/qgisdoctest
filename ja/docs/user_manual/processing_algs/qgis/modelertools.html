28.1.9.1. Calculate expression

NEW in 3.34

QGIS式の結果を計算し、同じ結果が複数回必要な場合に、モデル全体で同じ式を複数回使用する必要をなくします。さらに、他の方法では不可能なユースケースを可能にします。例えば、タイムスタンプ値を一度生成し、それをモデル内で複数回使用することができます。もしタイムスタンプが毎回再計算されるのであれば、その値はモデルのランタイム中に変化してしまうでしょう。

28.1.9.14. 可変距離バッファ

警告

このアルゴリズムは非推奨です。代わりに バッファ（buffer） アルゴリズムを使用してください。

入力レイヤにある全ての地物のバッファ領域を計算します。

与えられた地物のバッファサイズは属性で定義されるため、異なる地物が異なるバッファサイズを持つことができます。
