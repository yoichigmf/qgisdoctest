1.1. What is new in QGIS 3.34

This release of QGIS includes hundreds of bug fixes and many new features and enhancements, compared to QGIS 3.28. For a list of new features, visit the visual changelogs at https://qgis.org/en/site/forusers/visualchangelogs.html.

We recommend that you use this version over previous releases.
